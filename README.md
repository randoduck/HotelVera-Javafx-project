## Overview of the Project
This project was made as the final submission for the Object Oriented Software development Lab. The Techstack used was Java,JavaFX,Maven,SQLite,JDBC,Scene Builder,CSS. A Demo video will be uploaded very soon with improved UI.

The Hotel Booking System is a fully functional desktop application developed as a mini project for the Object Oriented Software Development Lab (OOSDL) course. The application simulates the core operations of a hotel front desk, allowing staff to manage rooms, create bookings, generate invoices and manage guests, all through an intuitive graphical interface.
The system is built using JavaFX for the graphical user interface, Maven as the build and dependency management tool, and SQLite via JDBC for persistent data storage. The application follows the Model- View-Controller (MVC) architectural pattern and implements all five additional features required for full marks.
